Java architecture for XML binding (JAXB)
JAXB stands for Java Architecture for XMLBinding and names an application programming interface (API)
for interaction between applications programmed in Java and XML documents.
JAXB is thus an alternative to well-known technologies such as Simple API for XML(SAX)
and Document Object Model( DOM), as it offers more convenient options for writing and reading XML documents
directly from the Java application.
